Teoria do CSS

O CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas HTML.
Com ele, podemos alterar cores, tamanhos, fontes, espaçamentos e posicionamentos dos elementos.
O HTML estrutura o conteúdo, e o CSS define como ele será exibido.
Vídeos Recomendados:

Como usar CSS?

Existem três formas principais de aplicar CSS em uma página:
Inline: diretamente dentro da tag HTML usando o atributo style.
Interno: dentro da tag <style> no <head>.
Externo: em um arquivo separado .css, linkado com <link rel="stylesheet" href="style.css">.
O método externo é o mais recomendado.

Seletores

Seletores definem quais elementos o CSS vai estilizar.
Exemplos:
• Seletor de tag: p { color: blue; }
• Seletor de classe(Atributo class=”titulo”): .titulo { font-size: 20px; }
• Seletor de id(Atributo id=”principal”): #principal { background: gray; }
Também é possível combinar seletores para maior precisão.

Cores

O CSS permite usar cores de várias formas:
• Nome da cor: color: red;
• Código hexadecimal: color: #ff0000;
• RGB: color: rgb(255, 0, 0);
• RGBA (com transparência): color: rgba(255, 0, 0, 0.5); 

                        .exemplo{
                            color: red;
                            background-color: rgba(255, 0, 0, 0.5);
                            border-color: #22222;
                            caret-color: rgb(0, 255, 0);
                        }

Fontes e Texto

Você pode alterar o tipo, tamanho, posição e cor do texto.
Exemplo: 

                        p {
                            font-family: Arial, sans-serif;
                            font-size: 16px;
                            text-align: center;
                            color: #333;
                        }

Lembre-se, você pode explorar as inumeras possibilidades de personalização do CSS, não se limite à apenas comandos simples.

Margem e Preenchimento (Margin e Padding)

Margin: espaço externo, fora do elemento.
Padding: espaço interno, entre o conteúdo e a borda.
Exemplo: 

                        div {
                            margin: 20px;
                            padding: 10px;
                        }

Bastante utilizado para dar um espaço entre conteúdos e organizar o site.

Bordas

O CSS permite personalizar bordas com cor, espressura, estilo e até arredondar.
Exemplo:

                        div {
                            border: 2px solid green;
                            border-radius: 10px;
                        }

Background

Define o plano de fundo de um elemento.
Você pode usar cores e imagens:
Exemplo:

                        body {
                            background-color: #f2f2f2;
                            background-image: url('imagem.jpg');
                        }
