Introdução ao Python

O Python é uma linguagem de programação de alto nível, de fácil leitura e escrita, criada por Guido van Rossum no final dos anos 1980. Ela é amplamente utilizada em diversas áreas, como análise de dados, inteligência artificial, automação, desenvolvimento web e muito mais. Sua principal característica é a simplicidade, tornando o Python uma das melhores opções para quem é iniciante.

Como instalar o Python

Para começar a usar o Python, siga os passos abaixo:
1 - Acesse o site oficial: https://www.python.org/downloads/
2 - Escolha a versão compatível com seu sistema operacional (Windows, macOS ou Linux).
3 - Após instalado, abra o terminal (ou prompt de comando) e digite:
python --version se aparecer versão do python, então a instalação foi bem sucedida.

Sintaxe Básica

O Python utiliza indentação (espaços) para definir blocos de código, em vez de chaves {} como em outras linguagens. Exemplo: 

                        if 5>2:
                            print(“Cinco é maior que dois”)

A indentação correta é essencial — ela define a estrutura do código.

Variáveis

As variáveis armazenam valores que podem ser usados depois. Não é preciso declarar o tipo, pois o Python detecta automaticamente. Exemplo: 

                        nome = “Ana”
                        idade = 25
                        altura = 1.65

Os nomes das variáveis devem ser claros e não começar com números.

Tipos de Dados

Os principais tipos são: int – números inteiros (10, -5) float – números decimais (3.14) str – texto ("Olá") bool – verdadeiro ou falso (True, False) Exemplo: 

                        x = 10
                        y = 3.5
                        nome = "João"
                        ativo = True

Entrada e Saída de Dados

A função print() mostra algo na tela. A função input() recebe dados do usuário. Exemplo: 

                        nome = input("Digite seu nome: ")
                        print("Olá,", nome)

Estruturas Condicionais

Servem para tomar decisões. Como o if e o else. Exemplo: 

                        idade = 18
                        if idade >= 18:
                            print("Você é maior de idade")
                        else:
                            print("Você é menor de idade")

Laços de Repetição

Permitem executar um bloco várias vezes. Exemplos são o for e o while.
Exemplo com for: 

                        for i in range(5):
                            print("Número:", i)

Exemplo com while:

                        contador = 0
                        while contador < 3:
                            print("Olá!")
                            contador += 1

Funções

Funções são blocos de código reutilizáveis. Para chamar uma função, basta colocar o nome atribuido a ela. Exemplo: 

                        def saudacao():
                            print("Olá")
                        saudacao()

Elas ajudam a organizar o código e evitar repetições.
