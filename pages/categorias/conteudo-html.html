Teoria do HTML

O HTML permite você criar o seu próprio Web Site.
Também é conhecido por ser fácil de se aprender entre todo tipo de programador.
Sendo um ótimo lugar para começar na programação caso seja um iniciante.
Abaixo estão alguns vídeos introdutórios para o HTML, recomendados por nós.

Onde posso escrever um código HTML?

Existe diversos lugares onde se pode começar, mas a principal IDE recomendada é o VisualStudio Code. Você pode instalar através do site oficial, seguindo esse link: https://code.visualstudio.com/

Estrutura Básica do HTML

Tudo começa com a declaração <!DOCTYPE html> e depois as tags principais:
<html>, <head> e <body>. Outras tags importantes podem ser usadas também
para o desenvolvimento do site.
O conteúdo visível da página sempre fica dentro de <body>, enquanto as
informações internas (como título e links de estilo) ficam dentro de<head>. 

                        <!DOCTYPE html>

                        <html lang="pt-br">

                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title> Nome do Website </title>
                        </head>

                        <body>
                        
                        </body>
                        </html>

Tags do HTML

As tags são os blocos de construção do HTML.
Elas indicam ao navegador o que cada parte do conteúdo representa.
Por exemplo, <p> define parágrafos, <h1> a <h6> definem títulos, e <a> cria links.
A maioria das tags tem abertura e fechamento: <p> texto </p>.

                        <body>
                            <h1>Título</h1>

                            <p>Parágrafo</p>

                            <a href="https://www.google.com/"> Link </a>
                        </body>

Atributos do HTML

Os atributos fornecem informações extras sobre uma tag.
Eles aparecem dentro da tag de abertura e seguem o formato atributo="valor".
Por exemplo: <img src="/imagem.jpg" alt="Descrição da imagem">.
No exemplo conseguimos ver um desses atributos, qual seria o [alt=”]
Os atributos ajudam a controlar o comportamento ou a aparência dos elementos.

Imagens no HTML

As imagens são inseridas com a tag <img>.
Ela é autofechada, e usa atributos como src (origem) e alt (descrição).
Exemplo: <img src="foto.jpg" alt="Minha foto de perfil">
O texto de <alt> aparece caso a imagem não carregue,
por não ter o caminho correto ou outro problema.

<img src="flores.png" alt="flores">

<img src="fllores.png" alt="flores">

Links no HTML

Os links são criados com a tag <a>.
O atributo principal é href, que define o destino do link.
Exemplo: <a href="https://www.exemplo.com">Visite o site</a>

<body>
    <a href="https://www.google.com/"> Visite o site </a>
</body>

Comentários no HTML

Comentários servem para deixar observações
dentro do código sem afetar o resultado visual.
Eles são muito úteis para lembrar o que certas partes fazem.
A sintaxe é: <!-- Isso é um comentário -->.

                        <body>
                            <h1>Título</h1>
                            <!--Isso não é lido pelo navegador-->
                            <p>Texto para auxiliar</p>
                        </body>
