Introdução ao JavaScript

JavaScript é uma linguagem de programação de alto nível, criada originalmente por Brendan Eich em 1995. Ela é amplamente utilizada no desenvolvimento web, sendo essencial para tornar páginas e aplicações interativas no navegador, além de também ser usada no backend por meio do Node.js, em apps mobile, automação e muito mais. Sua principal característica é a versatilidade, o que torna o JavaScript uma das melhores opções tanto para iniciantes quanto para desenvolvedores que buscam criar aplicações modernas e dinâmicas.

Como instalar o JavaScript

1 – Para começar a usar JavaScript (Node.js), siga os passos abaixo:
2 – Acesse o site oficial de downloads do Node.js: https://nodejs.org/pt/download
3 - Escolha a versão compatível com seu sistema operacional (Windows, macOS ou Linux).
4 – Após instalar, abra o terminal (ou Prompt de Comando) e digite: node --version
Se aparecer a versão do Node, a instalação foi concluída com sucesso.

Sintaxe Básica

A sintaxe é o conjunto de regras de escrita que o JavaScript usa para entender o código. Exemplo: 

                        let nome = "João Vitor"
                        const dataDeNascimento = “14/08/2009”

Variáveis

As variáveis são espaços na memória do computador.
Em JS existem 4 variações de como declarar uma variável, onde se utiliza-se o
padrão camelCase, entre elas estão: let, const, var e automaticamente.
let: pode mudar de valor;
const: não mudará de valor;
var: antiga, tem escopo de função e pode causar erros - má prática;
Automaticamente: cria variável global sem querer - má prática.
Exemplo: 

                        let a = 1
                        const b = 2
                        var c = 3
                        d = 4

Tipos de Dados

Os principais tipos são:
String - texto ("Olá")
Number - números inteiros e decimais (3; -5; 7.43)
Boolean - verdadeiro ou falso (True; False)
Null - reapresenta um valor vazio intencionalmente.
Undefined - indefinido (variável declarada, mas sem atribuição de um valor) 

                        let a = "Olá"
                        let b = 3
                        let c = false
                        let d = null
                        let e

Entrada e Saída de Dados

A função console.log() mostra algo na tela.
A função prompt() recebe dados do usuário.
Exemplo: 

                        let nome = prompt("Digite seu nome")
                        console.log(nome)

Estruturas Condicionais

Em JavaScript, a instrução if... else é uma estrutura fundamental de controle de
fluxo usada para executar diferentes blocos de código com base em uma condição específica.
Exemplo: 

                        a = "10"
 
                        if (a == "10") {
                            console.log("É 10")
                        } else {
                            console.log("Não é 10")
                        }

Laços de Repetição

São usados para executar um bloco de código várias vezes. Entre os laços de repetição mais usados estão enquadrados o for e o while.
Exemplo com for: 

                        for (let contador = 0; contador < 5; contador++) {
                            console.log(contador)
                        }

Exemplo com while:

                        let contador = 0
                        while (contador < 5) {
                            console.log(contador)
                            contador++
                        }

Funções

Servem para guardar blocos de código reutilizáveis.
Podem receber parâmetros e retornar valores.
Exemplo: 

                        function saudacao() {
                            console.log("Olá!")
                            }
                        saudacao()

Elas ajudam a organizar o código e evitar repetições.
