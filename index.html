Sobre o EncicloCode

O EncicloCode é uma ferramenta de aprendizado e de compartilhamento de códigos para ajudar os novatos que desejam iniciar nesta área de TI. Tendo suporte para linguagens como HTML5, CSS3, JavaScript e Python.

Princípios do EncicloCode:

    - Criar uma comunidade saudável: comunidade onde todos os usuários são iguais independente do conhecimento e origem. Sem fazer distinções, apenas focando na resolução das dúvidas ao decorrer dos estudos.

    - Conhecimento para todos: conhecimento deve ser para todos e não deve ser guardado, compartilhado, para que todos consigam evoluir no inicio de carreira na programação.

    - Segurança de dados: todos os dados são guardados sobre total sigilo em relação a informações que poderiam expor os usuários de maneira inadequada.

Objetivos:

    - Acesso ilimitado: acesso ao EncicloCode independente do lugar.

    - Crescimento constante: que nosso projeto se equipare a empresas como GitHub e Stack Overflow.

    - Aumento de conhecimento: usuários consigam evoluir em seus conhecimentos através dos estudos por compartilhamento dos códigos e aprendizado na área de conteúdo.
